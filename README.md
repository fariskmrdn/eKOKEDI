## Setup

### Install before run
```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

## Sass Compiler Location :
    /src/css


### Extention json setting :
add to vscode extention setting :

```sh 
"liveSassCompile.settings.formats":[

        {
            "format": "expanded",
            "extensionName": ".css",
            "savePath": "/src/css"
        }
    ], 

```
